NHS Crawler - Sam Wright

Web app for crawling and searching the NHS Choices website's conditions pages.

The API

  • /crawler/start to start crawling the website. This will take days to complete. You can stop the crawler and/or restart the app, then start the crawler again and it will resume from where it stopped. See the log output or /crawler/status to monitor its progress. Newly-crawled pages are added to the index for searching every 10 seconds.
  • /crawler/status to get the status of the crawler (as a json string of io.github.samwright.nhs.crawler.CrawlerStatus).
  • /crawler/stop to stop the crawler if it was running
  • /search/reindex to recreate the index from the crawled pages on disk. This can be called at any time, even while the crawler is running. This takes about a minute to complete, and its progress can be monitored at /search/status. You only need to do this if you copy json files yourself into the pages directory.
  • /search/status to get the status of the indexer (as a json string of io.github.samwright.nhs.search.IndexingStatus).
  • /search?q=mumps to search through the indexed pages for 'mumps' to get the best match, and return its NHS Choices URL.
  • /search/multiple?q=mumps to search through the indexed pages for 'mumps' to get multiple top matches (at most 10), and return them in order in a json string of a list of io.github.samwright.nhs.search.SearchResult.

Quickstart

  1. Click /crawler/start
  2. Click /search?q=mumps and try different queries. Newly-crawled pages will be searchable after at most a 10 second delay.
  3. Restart the app at any time and click /crawler/start to continue crawling from where it left off.

Options

  • maxPagesPerCrawl - maximum number of pages to crawl, useful for testing (default = -1, meaning no maximum).
  • crawlerCount - number of crawlers to run concurrently (default = 8).
  • dataDir - directory to store crawled pages, crawler metadata, and the lucene index. If not set, this will default to "${java.io.tmpdir}/io.github.samwright.nhs", unless you set...
  • randomiseTmpDir - if true and dataDir hasn't been set, store data in "${java.io.tmpdir}/io.github.samwright.nhs${randomLong}". Useful for testing.
  • deleteOnExit - if true, delete the data directory on exitting the app (default = false)
  • indexIntervalSeconds - the time to wait before adding newly-crawled pages to the index.

Command-line usage: java -jar nhs-crawler-0.2.0-SNAPSHOT.jar [--option=value]...

Author: Sam Wright
Uses: Apache Lucene, Spring Boot, Crawler4J, jsoup.
